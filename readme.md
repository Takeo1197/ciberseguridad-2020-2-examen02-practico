#Examen 02 (Parte Práctica)

## Instrucciones

- Clonar el repositorio
- Situarse dentro de la carpeta que se clonó
- Instalar dependencias del package.json ejecutando:
  ```
  npm install
  ```
- Ejecutar las preguntas respectivas de la siguiente manera:
  - Pregunta 1
    ```
    npm run question1
    ```
  - Pregunta 2
    ```
    npm run question2
    ```
  - Pregunta 3
    ```
    npm run question3
    ```
